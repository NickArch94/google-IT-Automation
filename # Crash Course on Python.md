# Crash Course on Python

### Syntax
- The rules for how instructions tend to be written.

### Semantics
- The effects of the instructions.

### Scripts vs Programmes:
**Scripts** are short/simple programs that can be run very quickly to run very specific tasks.
**Programmes** are fully-fledged programs that take longer development cycles to get properly functioning.

The process of **Automation** is replacing manual steps with ones that take place automatically.

### When is Automation to be used?
For large, time-consuming tasks that can be streamlined by computers. This frees up the individual to focus on a more intensive task.

## Why Python?
Python is easy to write, maintain, and understand. This makes writing code with much more beginner friendly. Also, Being available on many platforms, Python is able to be utilized on a wide range of different systems.
